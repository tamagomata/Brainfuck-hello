# Brainfuck-hello

```brainfuck
++++++++++[>+++++++>++++++++++>+++>+<<<<-]>++.>+.+++++++..+++.>++.<<+++++++++++++++.>.+++.------.--------.>+.>.
```

## コード解説

### 初期化部分
```brainfuck
++++++++++[>+++++++>++++++++++>+++>+<<<<-]
```
- `++++++++++` : セル0を10に設定（ループカウンタ）
- `[` : ループ開始
- `>+++++++` : セル1に7を加算
- `>++++++++++` : セル2に10を加算  
- `>+++` : セル3に3を加算
- `>+` : セル4に1を加算
- `<<<<-` : ポインタをセル0に戻してデクリメント
- `]` : セル0が0になるまでループ

**ループ終了後の各セルの値:**
- セル1: 10 × 7 = 70
- セル2: 10 × 10 = 100
- セル3: 10 × 3 = 30
- セル4: 10 × 1 = 10

### 「Hello」の出力
```brainfuck
>++.  // セル1: 70 + 2 = 72 → 'H'
>+.   // セル2: 100 + 1 = 101 → 'e'
+++++++..  // セル2: 101 + 7 = 108 → 'l' (2回出力)
+++.  // セル2: 108 + 3 = 111 → 'o'
>++.  // セル3: 30 + 2 = 32 → スペース
```

### 「World」の出力
```brainfuck
<<+++++++++++++++.  // セル1: 72 + 15 = 87 → 'W'
>.  // セル2: 111 → 'o'
+++.  // セル2: 111 + 3 = 114 → 'r'
------.  // セル2: 114 - 6 = 108 → 'l'
--------.  // セル2: 108 - 8 = 100 → 'd'
>+.  // セル3: 32 + 1 = 33 → '!'
>.  // セル4: 10 → 改行文字
```

## 出力結果
```
Hello World!
```
